POS System

The "POS System" is a versatile web application I've crafted, utilizing HTML, CSS, and JavaScript for the intuitive front-end interface. On the back end, it employs Spring Boot for robust functionality and MySQL as the database, ensuring seamless data management. This system empowers businesses to efficiently register customers, track items, and facilitate purchases for both registered and walk-in patrons. Its user-friendly design and powerful features demonstrate my proficiency in full-stack development, offering businesses a powerful tool to streamline their point-of-sale operations and enhance customer service.
